## How to use
All you have to do is run `$ ./setup.sh`.  
```
$ mkdir react-typescript-project && cd react-typescript-project  

! after you resolve path to setup command,
$ setup  

$ tree -I node_modules          
.
├── package-lock.json
├── package.json
├── src
│   ├── app.tsx
│   ├── home.tsx
│   ├── index.html
│   ├── index.tsx
│   └── subpage.tsx
├── tsconfig.json
└── webpack.config.js
 ```
